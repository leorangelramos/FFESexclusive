Teoria e Contexto em Ação: Verbos Modais (Should, Must)

Estes modais expressam níveis diferentes de obrigação ou recomendação.

1. SHOULD

Usado para dar conselhos ou recomendações. É uma sugestão, não uma ordem.

  • Conselho: You look tired. You should get some rest.
  • Recomendação: You should watch that movie. It's great!
  • Negativa (shouldn't): You shouldn't eat so much junk food.

2. MUST

Usado para expressar uma obrigação forte, necessidade ou uma conclusão lógica.

  • Obrigação: You must wear a seatbelt in the car. (É a lei)
  • Necessidade: I must remember to call my mom.
  • Conclusão lógica: He isn't answering the phone. He must be busy.
  • Negativa (mustn't) indica PROIBIÇÃO: You mustn't smoke in here.

1. Para dar um conselho: "It's a good book. You ____ read it."

a) should
b) must

2. Para uma regra/lei: "Passengers ____ show their tickets."

a) should
b) must

3. Para uma proibição: "You ____ touch that wire. It's dangerous."

a) shouldn't
b) mustn't

4. Para uma conclusão lógica: "The lights are off. They ____ be home."

a) mustn't
b) shouldn't

5. Para uma recomendação: "He ____ apologize for what he did."

a) should
b) must

1. Traduza a obrigação: "Eu tenho que ir agora."

2. Traduza o conselho: "Você não deveria se preocupar."

3. Traduza a proibição: "Nós não devemos fazer barulho."

1. To be healthy, you ____ eat more vegetables.

2. This is a secret. You ____ tell anyone.

3. She failed the test. She ____ study harder.

1. Ordene o conselho:

be / You / more / should / careful

2. Ordene a regra:

finish / work / We / must / today / this

3. Ordene a proibição:

not / library / You / must / in / talk / the
